# Water_pump_ZOE

> ⚠️ **Cette opération peut être très dangereuse pour votre véhicule** ; je ne serai pas tenu responsable des dommages que 
vous pourriez causer à votre voiture, alors soyez prudent si vous modifiez ce script.

Développement d'un logiciel permettant de **Lire/Reseter les compteurs de pompe à eau de la Zoe** au travers d'un outils 
de diagnostique ELM327 OBD2.

## Mes motivations

On trouve de nombreux articles sur Internet traitant de ce sujet. Cependant, ce projet est né du besoin d'avoir un 
**contrôle total sur le processus de réinitialisation**.

Plutôt que d'utiliser une "boîte noire" en passant par diverses applications tierces, cette solution propose une 
**alternative 100% gratuite et transparente**, basée sur l'exécution d'un seul script.
